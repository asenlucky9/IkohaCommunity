PLACE YOUR LOGO HERE
====================

Save your Ikoha Community logo image file as:

ikoha-logo.png

Or:
ikoha-logo.jpg
ikoha-logo.svg (recommended for best quality)

The logo will automatically appear in:
- Header (navigation bar)
- Homepage (hero section)
- Footer

Recommended logo size:
- Header: 150-200px width
- Homepage: 300-400px width
- Footer: 100-150px width

Formats supported: PNG, JPG, SVG
